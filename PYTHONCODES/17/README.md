# Quiz Game

## Description

A command-line trivia quiz game that presents multiple-choice questions and tracks player performance. Tests knowledge across various categories with immediate feedback.

## Features

- Question bank management
- Score tracking and reporting
- Check for remaining questions
- Final score calculation
- Feedback after completion

## Learning Outcomes

- Object-oriented programming principles
- Class design and implementation
- Data structure usage (lists, dictionaries)
- Game loop mechanics
- Score management

## How It Works

1. Creates question objects from question bank
2. Presents one question at a time
3. Evaluates user answers
4. Updates score for correct responses
5. Displays final score at completion

## Usage

```bash
python main.py
```

## Game Components

- Question: Data model for individual questions
- Quiz_Brain: Game engine and logic
- question_data: Question bank source

## Technologies Used

- Python 3
- Object-oriented programming
