# Coffee Machine Testing

## Description

Unit testing framework for the coffee machine project. Implements test cases for coffee machine functionality using Python's testing methodology.

## Features

- Unit tests for coffee operations
- Test case design
- Test execution framework
- Validation of machine behavior

## Learning Outcomes

- Test-driven development concepts
- Unit testing methodology
- Test case design
- Assertion and validation
- Coffee machine logic verification

## How It Works

1. Defines test cases for machine operations
2. Tests resource checking logic
3. Validates payment processing
4. Verifies drink preparation
5. Reports test results

## Usage

```bash
python main.py
python coffee_test.py
```

## Technologies Used

- Python 3
- Testing framework
- Unit testing concepts
