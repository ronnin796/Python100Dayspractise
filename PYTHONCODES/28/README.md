# Pomodoro Timer

## Description

A GUI-based Pomodoro timer application built with Tkinter. Implements the Pomodoro Technique with customizable work and break intervals. Tracks completed sessions with visual feedback.

## Features

- 25-minute work sessions
- 5-minute short breaks
- 20-minute long breaks after 4 work sessions
- Start and reset functionality
- Visual session counter
- Canvas-based countdown display

## Learning Outcomes

- GUI development with Tkinter
- Timer and scheduling mechanisms using after()
- Canvas widget for custom graphics
- Global state management
- Font styling and widget configuration

## Usage

```bash
python main.py
```

## Controls

- **Start**: Begin the current session timer
- **Reset**: Reset the timer and session counter

## Technologies Used

- Python 3
- Tkinter for GUI
- tkinter.font for custom text styling
