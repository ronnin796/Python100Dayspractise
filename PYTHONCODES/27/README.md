# Celsius to Fahrenheit Converter

## Description

A simple temperature conversion utility with a graphical user interface. Converts Celsius to Fahrenheit with input validation and error handling.

## Features

- Real-time temperature conversion
- Input validation with error messages
- Clean, intuitive GUI layout
- Support for decimal values

## Learning Outcomes

- GUI development with Tkinter
- Entry widget for user input
- Button command binding
- Exception handling for user input validation
- Label widget updates with dynamic content

## Usage

```bash
python main.py
```

## How It Works

1. Enter a temperature in Celsius
2. Click 'Convert' button
3. Result displays in Fahrenheit
4. Invalid input shows an error message

## Technologies Used

- Python 3
- Tkinter for GUI
