# Higher or Lower Game

## Description

A guessing game where players compare two celebrities or figures and predict which has higher social media followers. Features interactive gameplay with scoring system.

## Features

- Celebrity comparison game
- Follower count accuracy
- Score tracking
- Formatted data presentation
- Random selection of comparisons

## Learning Outcomes

- Game loop implementation
- Data structures (dictionaries, lists)
- Random selection
- Conditional logic
- Score management
- Data formatting

## How It Works

1. Two random figures are presented
2. Player guesses who has more followers
3. Correct guesses increase score
4. Game continues with new comparisons
5. Incorrect guess ends the game
6. Final score is displayed

## Usage

```bash
python main.py
```

## Game Components

- game_data: Contains celebrity and follower data
- Player guessing and comparison logic

## Technologies Used

- Python 3
- Random selection
- Dictionary/list operations
