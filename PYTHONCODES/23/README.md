# Frogger Game

## Description

A classic Frogger-style game built with Turtle graphics. Navigate a player across a busy road while avoiding moving cars, then reach the finish line.

## Features

- Player movement controls
- Obstacle course with moving cars
- Score/level tracking
- Game over detection
- Progressive difficulty increase
- Car generation and collision detection

## Learning Outcomes

- Turtle graphics for game development
- Object-oriented game architecture
- Collision detection algorithms
- Game loop implementation
- Event handling and keyboard input
- Animation and movement mechanics
- Sprite management

## How It Works

1. Player starts at bottom of screen
2. Use keyboard controls to move upward
3. Avoid moving cars on the road
4. Reach the top to complete level
5. Game increases difficulty each level
6. Collision ends the game

## Controls

- W: Move up
- Arrow keys or similar for movement

## Usage

```bash
python main.py
```

## Technologies Used

- Python 3
- Turtle graphics
- Object-oriented programming

## Game Components

- Player: Controlled character
- CarManager: Spawns and manages obstacles
- ScoreBoard: Tracks progress and levels
