# NATO Phonetic Alphabet

## Description

Converts words into NATO phonetic alphabet representation. Useful for clear communication over phone or radio where spelling clarity is critical.

## Features

- Word to NATO conversion
- Dictionary-based lookup
- Input validation
- Error handling for non-alphabetic characters
- Interactive command-line interface

## Learning Outcomes

- Dictionary comprehension
- CSV file parsing with Pandas
- DataFrame iteration
- Error handling and exceptions
- User input validation
- String operations and loops

## How It Works

1. Loads NATO phonetic alphabet from CSV
2. Prompts user for word input
3. Validates input contains only letters
4. Converts each letter to code word
5. Displays phonetic representation

## Usage

```bash
python main.py
```

## Example

Input: "HELLO"
Output: Hotel, Echo, Lima, Lima, Oscar

## Required Files

- nato_phonetic_alphabet.csv (alphabet mapping)

## Technologies Used

- Python 3
- Pandas for data handling
- Dictionary comprehension
