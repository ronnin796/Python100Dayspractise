# Guess the States

## Description

An interactive geography game where players guess all 50 US states on a blank map. Uses Turtle graphics for visualization and Pandas for state data management.

## Features

- Interactive state guessing game
- Visual feedback on guessed states
- Score tracking
- CSV export of missed states
- Exit option to save progress

## Learning Outcomes

- Turtle graphics for interactive games
- Pandas DataFrames and CSV operations
- Coordinate mapping and geometry
- Event handling (text input)
- Game loop implementation
- Data persistence with CSV

## How It Works

1. Blank map of US displays on screen
2. Player enters state names when prompted
3. Correctly guessed states are marked on map
4. Score updates with each correct guess
5. Missed states are saved to CSV on exit
6. Type 'exit' to quit and save progress

## Usage

```bash
python main.py
```

## Required Files

- blank_states_img.gif (map image)
- 50_states.csv (state coordinates and names)

## Technologies Used

- Python 3
- Turtle graphics
- Pandas
