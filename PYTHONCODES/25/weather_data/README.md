# Weather Data Analysis

## Description

Data analysis project using Pandas to process weather data. Calculates temperature statistics and performs data manipulation and filtering operations.

## Features

- Loads weather data from CSV
- Calculates average and maximum temperatures
- Temperature unit conversion (Celsius to Fahrenheit)
- Data filtering and row selection
- Creates custom DataFrames from dictionaries
- Data export to CSV

## Learning Outcomes

- Pandas DataFrame operations
- Data series methods (max, sum, to_list)
- Conditional filtering and selection
- Data type conversion and calculations
- DataFrame creation from dictionaries
- CSV export functionality

## How It Works

1. Loads weather_data.csv containing daily temperatures
2. Calculates average temperature across all days
3. Finds maximum temperature recorded
4. Converts Monday's temperature to Fahrenheit
5. Creates custom anime database
6. Exports results to CSV

## Usage

```bash
python main.py
```

## Required Files

- weather_data.csv (historical weather data)

## Output

- anime_data.csv (sample anime database)

## Technologies Used

- Python 3
- Pandas for data manipulation
