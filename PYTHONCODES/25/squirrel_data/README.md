# Squirrel Data Analysis

## Description

Data analysis project using Pandas to process squirrel observation data. Analyzes squirrel fur color distribution and generates summary statistics.

## Features

- Loads squirrel population data from CSV
- Extracts and counts fur color frequencies
- Generates summary report
- Exports analysis results to CSV

## Learning Outcomes

- Pandas DataFrame operations
- Data series manipulation
- Value counting and aggregation
- CSV file handling
- Data type extraction and filtering
- Set operations for unique values

## How It Works

1. Loads squirrel_data.csv dataset
2. Extracts unique fur colors
3. Counts frequency of each color
4. Generates summary statistics
5. Exports results to squirrel_count.csv

## Usage

```bash
python main.py
```

## Required Files

- squirrel_data.csv (raw observation data)

## Output

- squirrel_count.csv (frequency analysis)

## Technologies Used

- Python 3
- Pandas for data manipulation
