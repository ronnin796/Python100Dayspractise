# Python Games Collection

## Description

A collection of classic text-based and graphical games built with Python. Demonstrates fundamental programming concepts and game development techniques.

## Games Included

### Rock Paper Scissors (RPS.py)
- Classic RPS game against computer
- ASCII art display for choices
- Play again functionality
- Win/loss tracking

### Blackjack (blackjack.py)
- Casino blackjack simulator
- Dealer AI logic
- Hand scoring with Ace handling
- Hit/stand decisions

### Hangman (hangman.py)
- Word guessing game
- Stages display
- Letter tracking
- Win/loss conditions

### Other Games
- Caesar Cipher: Text encryption/decryption
- Guessing Number: Number prediction game
- Password Generator: Secure password creation
- Secret Auction: Blind bidding game

## Usage

Run individual games:
```bash
python RPS.py
python blackjack.py
python hangman.py
python ceasercipher.py
python guessingnum.py
python pypasswordgenerator.py
```
