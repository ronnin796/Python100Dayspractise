# Coffee Machine

## Description

An object-oriented coffee machine simulator with inventory management, payment processing, and drink preparation. Demonstrates core OOP principles including encapsulation and resource management.

## Features

- Multiple coffee drink options (Espresso, Latte, Cappuccino)
- Resource inventory tracking (water, milk, coffee)
- Payment processing and money management
- Drink validation before preparation
- Resource depletion tracking
- System report generation

## Learning Outcomes

- Object-oriented design principles
- Class design and responsibility separation
- Dictionary data structures
- Conditional logic for validation
- State management
- Resource constraint handling

## How It Works

1. Customer selects drink from menu
2. Machine checks resource availability
3. Payment is processed
4. Resources are deducted
5. Drink is "prepared"
6. Remaining resources are tracked

## Usage

```bash
python main.py
```
